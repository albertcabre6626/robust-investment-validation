# Data

This folder includes only public/non-sensitive data needed to reproduce the public modules.

## Included files

### `C5_public_returns.csv`

Public input file for the censored/public version of Module C5. It contains the observed returns used by the public C5 workflow.

- Rows: 1255
- Columns: 13
- Column names: `Date, C5_alpha_0.1, C5_alpha_0.2, C5_alpha_0.3, C5_alpha_0.4, C5_alpha_0.5, C5_alpha_0.6, C5_alpha_0.7, C5_alpha_0.8, C5_alpha_0.9, C5_alpha_1.0, Buy&Hold, Benchmark`

No private model logic or proprietary portfolio-construction code is included.
